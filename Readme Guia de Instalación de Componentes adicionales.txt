Para desarrollar este proyecto y poderlo ejecutar de manera correcta en tu computadora

después de hacer el git clone debes instalar los modulos necesarios para la edición y desarrollo de la misma

Componentes:

--Expo 
para instalarlo navega hasta la carpeta donde clonaste el proyecto

ejemplo Cd desktop/Proyect

e ingresa el siguiente comando: expo init AppC19

--Native Base
para instalarlo navega hasta la carpeta donde clonaste el proyecto

ejemplo Cd desktop/Proyect/AppC19

e ingresa el comando: npm install native-base

--Axios
para instalarlo navega hasta la carpeta donde clonaste el proyecto

ejemplo Cd desktop/Proyect/AppC19

e ingresa el comando: Expo install axios

--React - Navigation
para instalarlo navega hasta la carpeta donde clonaste el proyecto

ejemplo Cd desktop/Proyect/AppC19

e ingresa el comando: Expo install react-navigation
